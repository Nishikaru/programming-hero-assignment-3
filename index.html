What is the difference between grid and flexbox

Flexbox best for one dimensional Layout(like Row or Column) CSS grid best for 2D layout (Row and Column) Grid can do things Flexbox can't do, Flexbox can do things Grid can't do. They can work together: a grid item can be a flexbox container. A flex item can be a grid container. With CSS Grid we can set relationships horizontally and vertically but at the same time. Flexbox, on the other hand, is stuck doing either vertical or horizontal layouts.
